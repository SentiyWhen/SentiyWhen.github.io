# еә“ lib lib64 еӯҳж”ҫи·ҹвҪӮд»¶зі»з»ҹдёӯзҡ„зЁӢеәҸиҝҗвҫҸжүҖйңҖиҰҒзҡ„е…ұдә«еә“еҸҠеҶ…ж ёжЁЎеқ—гҖӮе…ұдә«еә“вјңеҸ«еҠЁжҖҒй“ҫжҺҘе…ұдә«еә“пјҢдҪңвҪӨзұ»дјјwindowsвҫҘзҡ„.dllвҪӮд»¶пјҢеӯҳж”ҫдәҶж №вҪӮд»¶зі»з»ҹзЁӢеәҸиҝҗвҫҸжүҖйңҖзҡ„е…ұдә«вҪӮд»¶гҖӮ

зі»з»ҹзҡ„еҮҪејҸеә“вҫ®еёёзҡ„еӨҡпјҢвҪҪ/libж”ҫзҪ®зҡ„еҲҷжҳҜеңЁејҖжңәж—¶дјҡвҪӨеҲ°зҡ„еҮҪејҸеә“пјҢд»ҘеҸҠеңЁ/bin жҲ–/sbinеә•дёӢзҡ„жҢҮд»Өдјҡе‘јеҸ«зҡ„еҮҪејҸеә“вҪҪе·І гҖӮ д»Җд№ҲжҳҜеҮҪејҸеә“е‘ўпјҹеҰіеҸҜд»Ҙе°Ҷд»–жғіжҲҗжҳҜеӨ–жҢӮпјҢ еҰӮжһңзҠ¶жҖҒдёәвҖңжңӘеҗҜвҪӨвҖқпјҢжҢүз…§еҰӮдёӢжӯҘйӘӨж“ҚдҪң 1гҖҒйҮҚеҗҜз”өи„‘пјҢеңЁдё»жқҝжҳҫзӨәз”»вҫҜпјҢеҝ«йҖҹеҜ»жүҫиҝӣвјҠBIOSзҡ„жҢүй”®гҖӮж №жҚ®е“ҒзүҢдёҚеҗҢпјҢеҸҜиғҪжҳҜF2гҖҒDel жҲ–е…¶д»–й”®гҖӮ 2гҖҒиҝӣвјҠBIOSеҗҺпјҢеҜ»жүҫиҝӣвјҠвҖңSystem ConfigurationвҖқгҖӮ 3гҖҒжүҫеҲ°вҖңVirtualization TechnologyвҖқпјҢжҢүеӣһв»Ӣй”®гҖӮ 4гҖҒйҖүжӢ©вҖңEnabledвҖқпјҢжҢүEnterеӣһв»Ӣй”®гҖӮ 5гҖҒз„¶еҗҺдҝқеӯҳйҮҚеҗҜеҚіеҸҜгҖӮ 2.2 е®үиЈ…иҝҮзЁӢ в»…зӣҙж’ӯи§Ҷйў‘ 2.3 й…ҚзҪ®иҝҮзЁӢ в»…зӣҙж’ӯи§Ҷйў‘ 3гҖҒ и®ӨиҜҶ Linux зҺҜеўғ LinuxдёӢзҡ„вҪ¬еҪ•йғҪжҳҜеҒҡд»Җд№ҲвҪӨзҡ„ /lib жҹҗдәӣжҢҮд»Өеҝ…йЎ»иҰҒжңүиҝҷдәӣеӨ–жҢӮжүҚиғҪеӨҹйЎәеҲ©е®ҢжҲҗзЁӢејҸзҡ„жү§вҫҸд№Ӣж„ҸгҖӮ е°Өе…¶йҮҚиҰҒзҡ„ жҳҜ/lib/modules/иҝҷдёӘвҪ¬еҪ•пјҢеӣ дёәиҜҘвҪ¬еҪ•дјҡж”ҫзҪ®ж ёвјјзӣёе…ізҡ„жЁЎз»„(й©ұеҠЁзЁӢејҸ)гҖӮ

иҝҷдёӘжҳҜз»ҷз¬¬дёүвҪ…еҚҸвј’иҪҜдҪ“ж”ҫзҪ®зҡ„вҪ¬еҪ• гҖӮ д»Җд№ҲжҳҜз¬¬дёүвҪ…еҚҸвј’иҪҜдҪ“е•ҠпјҹдёҫдҫӢжқҘиҜҙпјҢKDEиҝҷ дёӘжЎҢвҫҜз®ЎзҗҶзі»з»ҹжҳҜвјҖдёӘзӢ¬вҪҙзҡ„и®Ўз”»пјҢдёҚиҝҮд»–еҸҜд»Ҙе®үиЈ…еҲ°Linuxзі»з»ҹдёӯпјҢеӣ жӯӨKDEзҡ„иҪҜдҪ“ е°ұе»әи®®ж”ҫзҪ®еҲ°жӯӨвҪ¬еҪ•дёӢдәҶгҖӮ еҸҰеӨ–пјҢеҰӮжһңеҰіжғіиҰҒвҫғвҫҸе®үиЈ…йўқеӨ–зҡ„иҪҜдҪ“(вҫ®еҺҹжң¬зҡ„ distributionжҸҗдҫӣзҡ„)пјҢйӮЈд№Ҳд№ҹиғҪеӨҹе°ҶдҪ зҡ„иҪҜдҪ“е®үиЈ…еҲ°иҝҷвҫҘжқҘгҖӮ дёҚиҝҮпјҢд»ҘеүҚзҡ„Linuxзі»з»ҹ дёӯпјҢжҲ‘д»¬иҝҳжҳҜд№ жғҜж”ҫзҪ®еңЁ/usr/localвҪ¬еҪ•дёӢгҖӮ
